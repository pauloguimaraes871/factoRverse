# Constructor for step_winsorize

This is a low-level constructor for the `step_winsorize` preprocessing
step. It is called internally by
[`step_winsorize()`](https://pauloguimaraes871.github.io/factoRverse/reference/step_winsorize.md)
and is not intended to be used directly by users.

## Usage

``` r
step_winsorize_new(terms, role, trained, probs, winsor_limits, skip, id)
```

## Arguments

- terms:

  A list of quosures generated by
  [`rlang::enquos()`](https://rlang.r-lib.org/reference/enquo.html)
  specifying which columns to transform.

- role:

  Not used by this step as no new variables are created. Included for
  consistency with the `recipes` API.

- trained:

  A logical indicating whether the step has been trained. When `TRUE`,
  `winsor_limits` must be provided.

- probs:

  A numeric vector of length 2 indicating the lower and upper quantile
  probabilities used for winsorization.

- winsor_limits:

  A named list of length-two numeric vectors (per variable) with the
  computed quantile cutoffs (added during `prep()`).

- skip:

  A logical. Should the step be skipped when the recipe is baked by
  [`recipes::bake()`](https://recipes.tidymodels.org/reference/bake.html)?
  Defaults to `FALSE`.

- id:

  A character string that uniquely identifies the step.

## Value

An object of class `step_winsorize`.
