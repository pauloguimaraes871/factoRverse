# Accessor for Turnover Constraint Policy

Retrieves the turnover constraint policy from a `port_backtest_config`
object.

## Usage

``` r
get_turnover_constraint_policy(port_backtest_config_obj)

# S4 method for class 'port_backtest_config'
get_turnover_constraint_policy(port_backtest_config_obj)
```

## Arguments

- port_backtest_config_obj:

  A `port_backtest_config` object.

## Value

The turnover constraint policy list.
