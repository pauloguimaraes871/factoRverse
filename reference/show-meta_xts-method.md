# Show method for meta_xts

Shows a summary of the `meta_xts` object, including metadata and the
first few rows of the underlying `xts` data.

## Usage

``` r
# S4 method for class 'meta_xts'
show(object)
```

## Arguments

- object:

  An object of class `meta_xts`.

## Value

Returns the object invisibly after printing its summary.
