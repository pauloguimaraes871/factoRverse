# Accessor for Transaction Cost Parameters

Retrieves the transaction_costs_parameters from a `port_backtest_config`
object.

## Usage

``` r
get_transaction_costs_parameters(port_backtest_config_obj)

# S4 method for class 'port_backtest_config'
get_transaction_costs_parameters(port_backtest_config_obj)
```

## Arguments

- port_backtest_config_obj:

  A `port_backtest_config` object.

## Value

The turnover constraint policy list.
