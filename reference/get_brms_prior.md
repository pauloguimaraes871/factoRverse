# Get brms priors

Accessor function to retrieve brms priors.

## Usage

``` r
get_brms_prior(object)

# S4 method for class 'ss_backtest_results'
get_brms_prior(object)

# S4 method for class 'ss_backtest_config'
get_brms_prior(object)
```

## Arguments

- object:

  A `ss_backtest_config` or a `ss_backtest_results` object.

## Value

A `brmsprior` S4 class.
