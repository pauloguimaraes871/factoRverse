# Required Packages for step_winsorize

Declares the required packages for using the `step_winsorize` step
inside a recipe.

## Usage

``` r
required_pkgs.step_winsorize(x, ...)
```

## Arguments

- x:

  A `step_winsorize` object.

- ...:

  Additional arguments (not used).

## Value

A character vector of package names required for the step.
