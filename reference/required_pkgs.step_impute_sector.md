# Required Packages for step_impute_sector

Declares the required packages for using the `step_impute_sector` step
inside a recipe.

## Usage

``` r
required_pkgs.step_impute_sector(x, ...)
```

## Arguments

- x:

  A `step_impute_sector` object.

- ...:

  Additional arguments (not used).

## Value

A character vector of package names required for the step.
