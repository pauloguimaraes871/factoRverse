# Turn a concentration_constraint_policy into a list

Turn a concentration_constraint_policy into a list

## Usage

``` r
# S4 method for class 'concentration_constraint_policy'
as.list(x)
```

## Arguments

- x:

  An S4 object of class `concentration_constraint_policy`.

## Value

A named list containing:

- `benchmark`

- `max_abs_active_individual_weight`

- `max_abs_active_group_weight`
