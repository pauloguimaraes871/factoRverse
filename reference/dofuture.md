# dofuture operator

dofuture operator

## Usage

``` r
foreach %dofuture% expr
```

## Arguments

- foreach:

  object created by
  [`foreach::foreach()`](https://rdrr.io/pkg/foreach/man/foreach.html)
  and [`foreach::times()`](https://rdrr.io/pkg/foreach/man/foreach.html)

- expr:

  An R expression

## Value

The value of the foreach call..
