# Show Covariance Estimation Method

Displays the configuration of a `cov_est_method` object, including the
estimation method, the sample size used for the covariance matrix,
whether active returns are used, and the benchmark used.

## Usage

``` r
# S4 method for class 'cov_est_method'
show(object)
```

## Arguments

- object:

  A `cov_est_method` object.
