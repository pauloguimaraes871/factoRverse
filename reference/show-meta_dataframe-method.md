# Show Method for meta_dataframe Class

This method displays a summary of the `meta_dataframe` object, including
its name and current date, the signal (or target) columns and their
count, the unique dates and tickers, the total number of observations,
and the first few rows of the data.

## Usage

``` r
# S4 method for class 'meta_dataframe'
show(object)
```

## Arguments

- object:

  An instance of the `meta_dataframe` class.

## Value

The method returns the object invisibly.
