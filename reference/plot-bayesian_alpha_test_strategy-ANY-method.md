# Plot Bayesian Alpha Test Strategy Priors

Plots the prior distributions defined in a
`bayesian_alpha_test_strategy` object.

## Usage

``` r
# S4 method for class 'bayesian_alpha_test_strategy,ANY'
plot(x, y, ...)
```

## Arguments

- x:

  A `bayesian_alpha_test_strategy` object.

- y:

  Ignored. Included for S3 compatibility.

- ...:

  Additional arguments (currently unused).
