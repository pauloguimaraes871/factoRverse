# Accessor for Liquidity Floor Cutoffs

Retrieves the liquidity floor cutoffs from a `port_backtest_config`
object.

## Usage

``` r
get_liquidity_floor_cutoffs(port_backtest_config_obj)

# S4 method for class 'port_backtest_config'
get_liquidity_floor_cutoffs(port_backtest_config_obj)
```

## Arguments

- port_backtest_config_obj:

  A `port_backtest_config` object.

## Value

The liquidity floor cutoffs list.
