# Show Frequentist Alpha Test Strategy

Prints the contents of a `frequentist_alpha_test_strategy` object: a
header, followed by the inherited `alpha_test_strategy` display (main
information, model structure, and `lmer_control`).

## Usage

``` r
# S4 method for class 'frequentist_alpha_test_strategy'
show(object)
```

## Arguments

- object:

  A `frequentist_alpha_test_strategy` object to be displayed.
